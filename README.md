# Unity Kids Platformer 🍄

A 3D collectible platformer game developed with Unity 6 URP, designed for children on iPad/mobile.

## 🎮 Play It
[Play on itch.io](https://kaansener.itch.io/cocuk-oyunu)

## About
A mobile-first 3D platformer where players explore themed worlds and collect items. Built as a solo project to learn Unity game development from scratch.

**Completed Levels:**
- 🌲 Level 1 – Forest / Mushroom world
- 🏝️ Level 2 – Island / Fruit world

## 🛠️ Built With
- Unity 6 URP
- C#
- Cinemachine
- Unity Input System
- StarterAssets ThirdPersonController
- TextMeshPro
- Custom mobile joystick (MobileJoystick.cs)

## 📱 Platform
Primary target: iPad / Mobile (touch controls)
WebGL build available for browser play.
